how to run this script:
python3 decisionTrees.py <l> <k> training_set.csv validation_set.csv test_set.csv <to-print>

l,k are integer values used in pruning
to-print : {yes, no} this parameter decides to print the tree

Eg: python3 decisionTrees.py <l> <k> training_set.csv validation_set.csv test_set.csv <to-print>

Eg. python3 decisionTrees.py 40 45 training_set.csv validation_set.csv test_set.csv yes
